# AlquilaYa

![License](https://img.shields.io/badge/license-MIT-green)

AlquilaYa es una aplicación web desarrollada con React para la gestión de alquileres de propiedades. Permite a los usuarios buscar, filtrar y consultar detalles de propiedades en alquiler, ofreciendo una interfaz intuitiva y fácil de usar.

## Demo

Puedes ver la aplicación en funcionamiento aquí: [AlquilaYa](https://tu-url-de-demo-aqui.com) Se encuentra en proceso de desarrollo

## Características

- **Búsqueda de propiedades**: Busca propiedades en alquiler por ubicación, tipo y precio.
- **Filtros avanzados**: Filtra los resultados por diferentes criterios como precio, número de habitaciones y tipo de propiedad.
- **Detalles de propiedades**: Visualiza información detallada sobre cada propiedad, incluyendo fotos, descripción y contacto.
- **Responsive**: Optimizado para una experiencia fluida en dispositivos móviles y de escritorio.

## Tecnologías utilizadas

- **React**: Biblioteca de JavaScript para construir la interfaz de usuario.
- **API de Backend**: Conexión con un backend para obtener datos de propiedades (si aplica).
- **CSS Modules**: Estilos CSS locales para mantener el código modular y mantenible.

## Instalación

Para ejecutar este proyecto localmente, sigue estos pasos:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Jonavoe/AlquilaYa.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd AlquilaYa
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia la aplicación:
   ```bash
   npm start
   ```

La aplicación debería estar corriendo en `http://localhost:3000`.
